# Open the CFF initializer

Opens the CFF (Citation File Format) initializer web application in the
default browser.

## Usage

``` r
open_cffinit(...)
```

## Arguments

- ...:

  Unused arguments, for future extensibility.

## Value

Invisible `TRUE`, called for side effects.
